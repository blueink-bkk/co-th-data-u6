# O-ring type gaskets

Application : For immersion heaters, thermostats, temperature sensors 
or level sensors fittings, in NBR or silicone.  
Minimum size : 1"1/4.  
Maximum size : M77 x 2.  
Material : Silicone, NBR.  
Type : O-ring for rotation ring.
