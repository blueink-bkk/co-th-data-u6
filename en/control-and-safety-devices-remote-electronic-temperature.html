Remote electronic temperature control

Precise temperature control, LED display, one or two belts control.

Remote electronic temperature control

Main Features:

1. Standard electronic temperature control can be set in °C or °F, resolution 0.1° (for display of less than 100), on off action. Temperature set range -45 to + 120°C (-49 °F to 248 °F). Other temperature range up to 200°C (xx°F) on request. Protected against malicious set point modification.
2. Standard rating 16A 230V, or 20A 230V, 30A 115V with built in power relay version
3. Exclusive: One electronic control can be used to control 2 flexible belts located on the same drum.
4. 2 temperature sensing versions: -Built in sensor inside the belt -Remote sensor (No sensor inside the belt), for temperature sensing inside the drum. This sensor must always remain immersed inside the liquid.
5. Belt surface temperature protector is included if remote sensor
6. IP65, IK10, wall mounting control housing
7. 220/230V and 110/115V models
8. Includes a 3 meters long cord with connector, between controls and belt
9. Main power supply cord, oil resistant, available on request.
